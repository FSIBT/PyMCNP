Command Line Interface
======================

The `command line interface <../cli.rst>`_ (CLI) provides command line access to MCNP files. PyMCNP also includes a Python subpackage for using CLI subcommands programatically:

* ``pymcnp.cli``: Utilities for PyMCNP CLI.
